## 📖 Sobre o Projeto

A **Maga Soluções Industriais** atua oferecendo serviços especializados para o setor industrial, atendendo empresas que buscam eficiência operacional, segurança, 
confiabilidade e suporte técnico qualificado.

Este projeto foi desenvolvido como uma plataforma digital para fortalecer a presença online da empresa, apresentar seus serviços e facilitar o contato 
com clientes e parceiros.

---

## 🎯 Objetivos

- Apresentar a empresa e sua área de atuação
- Divulgar serviços industriais especializados
- Fortalecer a credibilidade da marca
- Facilitar a solicitação de orçamentos
- Melhorar a presença digital da empresa

## 🚀 Como Executar Localmente

### Clone o repositório

```bash
git clone https://github.com/Edvaldo-jb-Junior/maga-solucoes.git
```

### Entre na pasta do projeto

```bash
cd maga-solucoes
```

### Execute

Abra o arquivo:

```bash
index.html
```

ou utilize a extensão **Live Server** no VS Code.

### ⚙️ Manutenção Industrial
